Core idea

What is server-side scripting?

Server-side code runs on the web server after a request arrives, processes input, and then sends generated HTML back to the browser.

  • Handles forms, sessions, login systems, and business logic.
  • Can read files, upload documents, send emails, and access databases.
  • Keeps sensitive operations on the server instead of exposing them in the browser.
Request
Browser → Server
Processing
PHP executes logic
Response
HTML returned to user

PHP and HTML together

PHP is embedded inside HTML, so the server can generate dynamic content such as greetings, dates, form responses, and database results.

HTML page
<h1>Welcome</h1>
<?php echo "Dynamic page"; ?>
Benefits
  • Easy to connect forms with backend logic.
  • Useful for dynamic pages, login pages, and dashboards.
  • Supports direct communication with files and MySQL databases.

File handling in PHP

File handling lets PHP create, open, read, write, append, and check files stored on the server.

Common functions

  • fopen() – open a file
  • fread() – read file contents
  • fwrite() – write data to a file
  • fclose() – close an open file
  • file_exists() – check availability
<?php
$file = fopen("notes.txt", "w");
fwrite($file, "PHP file handling demo");
fclose($file);
?>

File uploading workflow

Steps

  1. Create an HTML form with method="post".
  2. Set enctype="multipart/form-data".
  3. Use file input and submit button.
  4. Process the file using the $_FILES array.
  5. Validate name, type, size, and move the file to a target folder.
<form action="upload.php" method="post"
      enctype="multipart/form-data">
  <input type="file" name="myfile">
  <input type="submit" value="Upload">
</form>
<?php
move_uploaded_file(
  $_FILES["myfile"]["tmp_name"],
  "uploads/" . $_FILES["myfile"]["name"]
);
?>

Email basics in PHP

What email needs

  • Recipient address
  • Subject line
  • Message body
  • Headers such as From, Reply-To, MIME-Version, and Content-Type
<?php
$to = "student@example.com";
$subject = "PHP mail demo";
$message = "This email is sent from PHP";
$headers = "From: admin@example.com";
mail($to, $subject, $message, $headers);
?>

Email with attachments

To send an attachment, PHP builds a multipart MIME email, encodes the file with base64, and places it inside the message body.

📎

Attachment file

Read the file from local storage using file functions.

🔐

Base64 encoding

Convert binary content into email-safe text format.

🧾

MIME boundary

Separate the HTML body and the attached file inside one email.
