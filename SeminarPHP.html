What is server-side scripting?

  • PHP code runs on the web server before the browser receives the page.
  • The server can process input, apply logic, and interact with a database.
  • The browser usually receives the final output as plain HTML.
  • This makes web pages dynamic, personalized, and data-driven.

PHP inside HTML

Idea

PHP can be embedded directly in an HTML document using <?php ... ?> tags. This lets developers mix static markup with dynamic values.

Typical use cases include showing the current date, printing variables, loading user names, and deciding which content block to display.

<!DOCTYPE html> <html> <body> <h1>Welcome</h1> <p>Today is <?php echo date("l"); ?> </p> </body> </html>

Simple PHP script: Hello World

<?php // first program echo "Hello, World!"; ?>

How it works

  • The file is saved with the .php extension.
  • The server identifies the file and interprets the PHP code.
  • echo sends output into the HTML response.
  • The browser displays the returned text as part of the page.

Variables and expressions

🔤

Variables

PHP variables start with $, such as $name or $age.

🧮

Operators

You can add numbers, join strings, compare values, and build logic using operators.

📤

Output

Computed values can be printed using echo or inserted in HTML.

<?php $name = "Ravi"; $marks = 85; echo "Student: " . $name; echo "<br>"; echo "Marks: " . $marks; ?>

Control structures

Decision making

Conditions let the server choose different output for different inputs. This is useful for login status, grades, and form validation.

<?php $score = 78; if ($score >= 50) { echo "Pass"; } else { echo "Fail"; } ?>

Repetition

Loops generate repeated HTML such as list items, tables, and menu links.

<?php for ($i = 1; $i <= 5; $i++) { echo "Item " . $i . "<br>"; } ?>

Handling HTML forms

<form method="post" action="welcome.php"> Name: <input type="text" name="username"> <button type="submit">Send</button> </form>

Processing input

<?php $username = $_POST['username']; echo "Welcome, " . $username; ?>

Forms send data to PHP using GET or POST. The server can read the submitted values, validate them, and create a response page.

Connecting PHP to MySQL

Common options

  • MySQLi procedural style
  • MySQLi object-oriented style
  • PDO for flexible database access

In labs and local practice, students often run Apache and MySQL through XAMPP before testing PHP pages.

<?php $conn = new mysqli("localhost", "root", "", "college"); if ($conn->connect_error) { die("Connection failed"); } echo "Connected successfully"; ?>

Best practices for beginners

  • Keep PHP, HTML, and SQL code neatly organized.
  • Validate form data before saving it.
  • Reuse connection code using include files or functions.
  • Show helpful error messages during learning, but hide sensitive details in real applications.
  • Move gradually from simple examples to full CRUD projects.
